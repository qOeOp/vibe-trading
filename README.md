# Vibe Trading

一个基于 TradingView 的交易行情展示应用。

## 技术栈

- Vite
- TypeScript
- Docker + Nginx

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## Docker 部署

```bash
# 使用 docker-compose 启动
docker-compose up -d
```

应用将在 `http://localhost:80` 运行。
