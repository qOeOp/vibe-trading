## 北向资金解读框架

### 北向资金的信号价值

- 北向资金 = 外资通过沪/深股通买入 A 股
- 被视为"聪明钱": 信息优势 + 更长投资视野
- **局限**: 2024年后北向实时数据披露规则变化 (延迟公布), 信号价值有所下降

### 关注维度

1. **总量**: 大幅净买入/净卖出 → 外资对A股整体态度
2. **行业偏好变化**: 增持哪些行业 → 外资配置方向
3. **个股集中度**: 单日大额买入单一标的 → 可能有公司层面催化

### 与其他资金指标的区别

| 指标 | 代表谁 | 信号含义 |
|---|---|---|
| 北向资金 | 外资 | 长线配置方向, 对汇率敏感 |
| 主力资金 | 机构+游资 | 短线操盘方向 |
| 融资余额 | 杠杆资金 | 市场情绪温度计 |
| 龙虎榜 | 知名机构/游资 | 个股事件驱动 |

## 主力资金分析注意事项

### 数据局限性

- 主力资金按"单笔成交金额"分级, **不等于**真实机构资金
- 机构可能拆单 (将大单拆成多个小单), 导致统计失真
- 更可靠的机构资金追踪: 北向资金 (实名制) > 龙虎榜 (有席位信息) > 主力资金流 (推算)

### 融资融券分析框架

- 融资余额持续上升 + 市场上涨 → 牛市信号 (杠杆资金入场)
- 融资余额持续上升 + 市场横盘 → 风险累积 (杠杆高位)
- 融资余额快速下降 → 去杠杆, 可能引发踩踏
- 融券数据在 A 股参考价值有限 (融券标的和券源有限)

### 市场温度计 (Fear/Greed)

**综合指标**: 0-100 分, 仪表盘展示

| 分位 | 状态 | 含义 |
|---|---|---|
| 80-100 | 极度贪婪 | 逆向减仓信号 |
| 60-80 | 贪婪 | 市场偏热 |
| 40-60 | 中性 | 正常 |
| 20-40 | 恐惧 | 市场偏冷 |
| 0-20 | 极度恐惧 | 逆向加仓信号 |

**组成维度** (各 20%):
1. 北向资金方向 (10日趋势)
2. 融资余额趋势 (20日)
3. 涨跌比 (上涨家数/总家数)
4. 换手率偏离 (vs 60日均值)
5. 波动率 (20日年化波动率 vs 历史分位)

## 龙虎榜分析框架

### 席位分类

| 类型 | 特征 | 信号含义 |
|---|---|---|
| 机构专用 | 显示"机构专用" | 中长线布局, 可能有基本面逻辑 |
| 知名游资 | 赵老哥/炒股养家等 | 短线操盘, 关注接力 |
| 量化席位 | 中信/华泰等大量交易 | T+0 策略, 参考价值低 |
| 外资席位 | 沪股通/深股通 | 配合北向资金分析 |

### 龙虎榜策略参考

- 机构净买入 + 行业景气 → 中线关注
- 游资接力 (多个游资买入) → 短线强势, 但需警惕退潮
- 机构卖出 + 高位放量 → 兑现信号

## 市场温度计计算方法

每个维度独立计算 0-100 分, 等权平均:

1. **北向资金**: 10日累计净流入在过去1年的分位数
2. **融资余额**: 融资余额20日增速在过去1年的分位数
3. **涨跌比**: 今日上涨家数/总家数, 映射到 0-100
4. **换手率偏离**: 全市场换手率 / 60日均值, >1.5 偏热, <0.7 偏冷
5. **波动率**: 20日年化波动率在过去1年的分位数 (反向: 高波动=恐惧)
