## 政策周期状态

当前政策立场判断:
- 🟢 宽松: 降准/降息周期, 流动性充裕
- 🟡 观望: 政策空窗期, 方向未明
- 🔴 收紧: 加息/缩表预期, 流动性收紧

判断依据: MLF 利率趋势 + 逆回购量 + 官方表态

## 指标计算说明

**PMI**: 制造业采购经理指数, 50为荣枯线
- >50: 经济扩张
- <50: 经济收缩
- 关注子项: 新订单、新出口订单、就业

**CPI**: 居民消费价格指数, 同比增速
- >3%: 通胀压力较大
- <0%: 通缩风险
- 核心CPI: 剔除食品和能源, 更反映趋势

**社融**: 社会融资规模增量
- 超预期: 信用扩张, 利好银行/地产
- 不及预期: 信用收缩, 利空经济敏感板块

## 财报季交易策略参考

| 策略 | 时机 | 逻辑 |
|---|---|---|
| 预告超预期买入 | 1月业绩预告发布后 | 业绩超预期个股通常有 2-4 周超额收益 |
| 年报高送转预期 | 3月 | 高送转概念在年报披露前有预期差 |
| 中报验证 | 8月 | 验证上半年增速是否持续, 调仓依据 |
| 三季报布局 | 9-10月 | 全年业绩基本确定, 布局确定性高的标的 |

> 注意: 以上为历史规律参考, 非投资建议

## 解禁减持数据源

- 限售解禁日历: AKShare `stock_restricted_release_queue_sina()`
- 重要股东减持: AKShare `stock_share_change_cninfo()`
- 影响评估: 解禁市值 / 流通市值 > 10% 时标注"大额解禁"
