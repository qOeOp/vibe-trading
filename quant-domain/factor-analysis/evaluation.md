# 因子评价 (Factor Evaluation)

## 1. 稳健回归 (Robust Regression / RLM)

传统 OLS 对离群值敏感。RLM 通过迭代赋权回归 (IRLS) 减小异常值影响。
- **应用**: A 股截面回归单因子测试的标准做法。

## 2. 关键统计指标

| 指标 | 含义 |
|---|---|
| **Mean IC** | 回测期内的平均 IC |
| **IC T-Stat** | 衡量 Mean IC 是否显著异于零 |
| **IC > 0 占比** | 因子预测方向正确的周期百分比 |
| **IC 衰减 (Decay)** | 随预测跨度增加（T+1, T+5, T+20），IC 下降的速度 |

## 3. 进阶分析

### 分层分析 (Quantile Analysis)
将样本池分成 N 个桶（如十分位），比较多头（Q1）与空头（Q10）的累计收益。单调递增/递减的分层曲线是因子有效性的直观证据。

### IC 行业分布
识别因子在哪些行业表现最为出色，指导行业中性化或行业定制化策略。

### 因子动物园问题 (Factor Zoo)
测试过多因子时，"假阳性"概率增加。
- **缓解**: 提高新因子准入门槛 — **T-stat ≥ 3.0 是底线**
- **方法论**: Feng, Giglio, Xiu (2020) 提出 Double-LASSO 方法，在控制已知因子后检验新因子的增量贡献
- **参考**: [references.md #8] "Taming the Factor Zoo"

### 多重检验校正：LFDR 优于 FDR

- **FDR**: 针对整体门槛的平均假阳性率（"所有显著因子中有多少比例是假的"）
- **LFDR (局部假发现率)**: 针对单个观测值的后验概率（"这个 T=3.5 的因子有多大可能是假阳性"）— 更直接、更个性化
- **关系**: FDR(τ) = LFDR 在所有 |T|≥τ 观测值上的均值；通常 LFDR(τ) ≥ FDR(τ)
- **实用解读**: LFDR=0.2 意味着支持 H₁ 的概率是 H₀ 的 4 倍

**关键陷阱 — 因子间相关性**:
- 多数校正方法假设各检验独立，但因子间共享宏观驱动因子导致相关性
- 相关性引起零假设分布的均值偏移 + 方差膨胀 → 标准正态假设下严重低估假阳性率
- Bootstrap 无法修复此问题（平均后看似正态，但单次抽样仍带系统偏移）
- **解决**: 使用经验零假设分布估计（Efron 2007），而非假设标准正态

**建议**: 对每个待检验因子报告 LFDR，而非仅报告 p 值。T-stat ≥ 3.0 是考虑相关性和多重检验后的最低门槛。

## 4. 夏普比率推断 (Sharpe Ratio Inference)

> 来源: López de Prado, Lipton, Zoonekynd (2026) "How to Use the Sharpe Ratio" — [references.md #1]

### 概率夏普比率 (PSR)
在给定基准 $SR_0$ 下，计算 $\widehat{SR}$ 真正超过 $SR_0$ 的概率。

**采样分布标准差**:
$$\sigma[\widehat{SR}] = \sqrt{\frac{1}{T} \left( \frac{1+\rho}{1-\rho} - \gamma_3 \widehat{SR} + \frac{\gamma_4-1}{4} \widehat{SR}^2 \right)}$$

- $T$: 样本长度
- $\rho$: 收益率自相关系数
- $\gamma_3, \gamma_4$: 偏度、峰度

**推论**: PSR 惩罚负偏度（下行风险）、正超额峰度（肥尾）和正序列相关（波动率聚集）的策略。

### 最小业绩记录长度 (MinTRL)
在显著性水平 $\alpha$ 下，证明 $\widehat{SR}$ 显著优于 $SR_0$ 所需的最短时间。$\widehat{SR}$ 越接近 $SR_0$，或收益偏离正态越严重，MinTRL 越长。

### 减偏夏普比率 (DSR)
修正"回测过拟合"和"多重测试"导致的虚高 SR。
- $K$: 独立测试总次数
- $V[\{\widehat{SR}_k\}]$: 所有尝试策略 SR 的横截面方差
- 尝试 $K$ 个因子后选最优，DSR 根据尝试广度对最高 SR "脱水"

### 贝叶斯发现率
- **pFDR (Planned)**: 部署前根据测试功效预估的虚假发现概率
- **oFDR (Observed)**: 观测到特定 SR 后，该策略属于"噪声"的后验概率

### 使用场景
- **FWER (如 DSR)**: 决策具有系统性影响时（如公司级资本分配），严控第一类错误
- **FDR**: 决策局部且可分散时（如量化多因子库准入），平衡预测力与准确性

## 5. 鲁棒性检验 (Robustness Tests)

> 因子通过统计显著性检验后，还需验证其预测能力是否依赖于数值精度的"巧合"，而非真实的经济逻辑。

### Bootstrap 置信区间

通过 Bootstrap 重采样模拟多种市场路径，检验因子表现的稳定性：

| 指标 | 方法 | 警戒标准 |
|---|---|---|
| **净值曲线置信区间** | 每个时间点 T 的 B 次样本净值排序，取中间 95% 范围 | 原始净值曲线超出上界 → 过拟合风险高 |
| **Sharpe P5 下界** | Bootstrap Sharpe 分布的第 5 百分位值 | P5 > 0.5 → 多数极端情景仍盈利；P5 < 0 → 不可靠 |
| **最大回撤 P95** | Bootstrap 所有样本的最大回撤分布 | $MDD_{worst} > -30\%$ → 可控；$MDD_{worst} < -50\%$ → 危险信号 |

**与 PSR 的关系**: PSR 是解析解（假设收益分布已知），Bootstrap 是非参数方法（不假设分布形态）。两者互补——PSR 更精确但依赖分布假设，Bootstrap 更稳健但需要足够样本量。

### Rank Test 与 Binary Test

检验因子的预测能力是否源于**相对排序**而非**绝对数值**：

| 检验 | 变换 | 含义 |
|---|---|---|
| **Rank Test** | $X \to \text{rank}(X)$ | 将因子值替换为截面排名，保留序数信息 |
| **Binary Test** | $X \to \text{sign}(X)$ | 将因子值替换为正负号，仅保留方向信息 |

**判定规则**:
- 变换后 Sharpe / $R^2$ 下降 < 30% → 因子逻辑扎实，捕捉的是结构性信号
- 变换后下降 > 70% → 因子严重依赖数值精度，过拟合风险极高
- Rank Test 通过 + Binary Test 通过 → 因子同时具备排序稳定性和方向稳定性

**与预处理 rank 算子的关系**: `preprocessing.md` §5 的 rank→power→right_tail 是**信号后处理**（改善权重分布），此处的 Rank Test 是**因子验证**（检验是否过拟合）。目的不同但算子相同。
