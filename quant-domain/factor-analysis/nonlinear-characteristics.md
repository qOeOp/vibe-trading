# 非线性特征处理 (Nonlinear Characteristics)

## 现象

因子与股票收益之间并非严格线性。

### 典型表现
- **PE**: 并不是越小越好，"较小"的最好（中间位置最优）
- **两端掉头**: 因子整体呈线性，但在极好或极差端出现非线性变化
- **换手率**: 在极低端可能出现预测方向的反转

## 处理方式

| 方法 | 说明 |
|---|---|
| **多项式变换** | 使用因子的多项式形式（$X$, $X^2$, $X^3$）进行预测 |
| **分段线性化** | 对因子分桶后在每个桶内建立独立的线性关系 |
| **附加因子** | 引入市值等作为附加维度，对特定样本空间（如小市值）单独建模 |

## 实践建议

- 在单因子测试中，观察分层曲线（Quantile Plot）是否单调——非单调即非线性信号
- 对非线性因子，优先尝试分桶处理而非直接回归
