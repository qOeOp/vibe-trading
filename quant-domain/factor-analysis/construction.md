# 因子构建 (Factor Construction)

## 1. 定义

**因子 (Factor)** 是用于筛选股票和产生超额收益 (Alpha) 的定量指标。代表了从股票特征（价格、成交量、基本面数据）到预期收益或风险概况的映射。

## 2. 按依赖关系分类

| 类型 | 定义 | 示例 |
|---|---|---|
| **基础因子 (Leaf)** | 直接从原始数据衍生 | `close`, `PE`, `total_assets` |
| **复合因子 (Composite)** | 使用算子对基础因子转换或组合 | `rank(PE) / N`, `SMA(close, 20)` |

复合因子进一步区分：
- **截面因子 (CS)**: 值取决于样本池中其他股票（如 `rank(PE) / N`）
- **序列因子 (TS)**: 值仅取决于单只股票的历史数据（如 `SMA(close, 20)`）

## 3. 构建逻辑：位置 vs 速度 vs 加速度

不应仅停留在当前值（位置），还应考虑变化率（速度）和变化稳定性（加速度）。

| 概念 | 含义 | 示例 |
|---|---|---|
| **位置 (Position)** | 因子当前暴露值 | $ROE$ |
| **速度 (Velocity)** | 一阶导数/同比变化 | $\Delta ROE$ |
| **加速度 (Acceleration)** | 二阶导数 | $\Delta(\Delta ROE)$ |

*速度和加速度往往能提供独立于位置的增量预测信息。*

## 4. 序列算子 (TS Operators)

应用于单只证券的历史向量。

| 算子 | 含义 |
|---|---|
| `REF(X, n)` | n 期前 X 的值（滞后） |
| `DELTA(X, n)` | $X - REF(X, n)$（差分） |
| `MA(X, n)` | 简单移动平均 |
| `EMA(X, n)` | 指数移动平均 |
| `STD(X, n)` | 标准差 |
| `HHV(X, n)` / `LLV(X, n)` | n 期内最高/最低值 |
| `TS_ZSCORE(X, n)` | 时间序列 Z-Score |

## 5. 截面算子 (CS Operators)

应用于同一时间点样本池内所有证券。

| 算子 | 含义 |
|---|---|
| `RANK(X)` | X 在样本池中的百分比排名 [0, 1] |
| `DEMEAN(X)` | 减去截面均值 |
| `CS_ZSCORE(X)` | 截面 Z-Score |
| `QUANTILE(X, n)` | 按 X 值分成 n 个分桶 |
| `INDUSTRY_NEUTRALIZE(X)` | 行业中性化（减去行业均值） |
