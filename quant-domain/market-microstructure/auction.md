# 竞价机制 (A 股 Auction)

## 集合竞价 (Call Auction)

| 时段 | 规则 | 备注 |
|---|---|---|
| 09:15 - 09:20 | 可报单可撤单 | 价格发现，注意"虚假"单 |
| 09:20 - 09:25 | 可报单**不可**撤单 | 反映真实开盘意向 |
| 14:57 - 15:00 | 收盘集合竞价（仅深交所） | 确定收盘价 |

### 撮合原则
1. 成交量最大
2. 最小剩余未成交量
3. 最接近前收盘价

## 连续竞价 (Continuous Auction)

| 时段 | 说明 |
|---|---|
| 09:30 - 11:30 | 上午 |
| 13:00 - 14:57 (深) / 15:00 (沪) | 下午 |

**原则**: 价格优先 → 时间优先

## 订单类型

| 类型 | 说明 |
|---|---|
| **LMT** | 按固定价格报单 |
| **MKT** | 按当前最优价格立即成交 |
| **FAK** | 立即尽可能多成交，撤销剩余 |
| **FOK** | 全部成交否则整单撤销 |
| **最优五档** | 针对订单簿前五档撮合 |

## 流动性观察

- **流动性真空**: 11:30-13:00 停盘 → 下午开盘价格跳空
- **竞价尖峰**: 开盘/收盘前波动率较高
- **订单不平衡**: 集合竞价买卖量差异可作为短期情绪指标
