# 高频因子 (High-Frequency Factors)

利用日内高频数据（5 分钟或 Tick）刻画日内特征，提供日频数据之外的增量信息。

## 1. 非流动性因子 (Illiquidity)

| 因子 | 公式/含义 |
|---|---|
| **ILLIQ** | $Mean(\|Return\| / Volume)$ |
| **SILLIQ** | 卖单触发的非流动性 |
| **BILLIQ** | 买单触发的非流动性 |

*流动性风险具有独立于市值效应之外的收益预测能力。*

## 2. 收益分布特征

| 因子 | 含义 |
|---|---|
| **特质偏度** | 收益分布的非对称性 |
| **特质峰度** | 收益分布的肥尾特征 |
| **DUVOL** | 上下行波动率比，衡量价格波动不对称性 |

## 3. 常用高频指标

| 因子 | 含义 |
|---|---|
| **VSTD** | 成交量波动率，与股价波动率低相关 |
| **BSI** | 买卖不平衡，衡量资金流向 |
| **NCSKEW** | 负偏度系数，预测暴跌概率 |
| **VPIN** | 成交量同步知情交易概率，衡量流毒性 |

## 4. 订单簿指标

| 指标 | 公式 |
|---|---|
| **OIB** | $(BidVol - AskVol) / (BidVol + AskVol)$ |
| **Spread** | $Ask_1 - Bid_1$ |
| **VPIN** | 衡量流动性压力 |
