# 行业特有因子 (Industry-Specific Factors)

针对不同行业独特的经济逻辑，寻找在该板块内具有更强解释力的特定指标。

## 1. 行业因子图谱

| 行业 | 关键因子/逻辑 |
|---|---|
| **农林牧渔** | EP、盈利能力增长。行业具有资源稀缺性，盈利好转趋势对个股影响巨大。 |
| **采掘 (煤炭/石油)** | 派息率、现金流强度。市场更青睐现金充裕且高分红的标的。 |
| **化工** | 低成交量、低换手、小市值。投资者倾向于交易冷清的小盘股。 |
| **黑色金属 (钢铁)** | 流动负债比例。高流动负债通常证明企业对上游议价能力强，能获得更多流动资金。 |
| **有色金属** | 市销率 (PS)、长期负债。低 PS 且长期负债较低的个股更受看好。 |
| **建筑建材** | 毛利率。行业技术门槛低，毛利率高证明其盈利能力和成本控制能力强。 |
| **机械设备** | 低市净率 (PB)。具有阶段性价值回归潜力。 |
| **电子** | 低成交量、低换手。冷门股在该行业内表现往往更好。 |
| **交通运输** | 资产周转率。运营效率是核心，周转越快质地越好。 |
| **家用电器** | PE 估值、高流动负债。高流动负债反映了对上游的话语权。 |
| **食品饮料** | 营运能力、负债结构。周转到位下的高负债能有效降低财务成本。 |
| **医药生物** | 六个月价格反转。医药股具有较强的中长期反转特征。 |
| **银行业** | 贷存比 (LDR)、信用成本 (Credit Cost)。分别从“量”和“质”上考察主营业务。 |
| **房地产** | 估值因子 (PE/PB)。受政策调控影响大，政策松动时低估值股票先行。 |

## 2. 研究思路
1. **统计分析**: 在行业样本池内进行单因子有效性检验（IC/IR/胜率）。
2. **经济意义**: 验证因子方向是否符合行业运行的逻辑。
3. **行业中性化**: 在全市场模型中，通过扣除行业均值或回归残差来应用这些特有信息。
