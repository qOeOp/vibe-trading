# Thread: Purpose

Project goals and problem statement for the treemap preview application.

---

## Task: [Project Purpose](./tasks/01-project-purpose.md)

**Goal:** Build interactive treemap visualization for Chinese stock market sectors  
**Problem:** Complex hierarchical data needs intuitive visual representation  
**Solution:** Drill-down treemap with glassmorphism design

---

## Key Objectives

1. **Visualize Hierarchy:** 4-level drill-down (Sector → Industry → Sub → Stock)
2. **Show Data:** Capital flow, price change, attention level per entity
3. **Enable Navigation:** Click to drill-down, breadcrumb to navigate up
4. **Search:** Cross-level entity search
5. **Modern UI:** Glassmorphism, 3D hover, smooth animations

---

## Target Users

- **Traders:** Quick sector performance overview
- **Analysts:** Hierarchical market structure exploration
- **Investors:** Capital flow and attention tracking
