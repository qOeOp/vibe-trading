# Task: Container Specs

Fixed dimensions for HeatMapContainer and internal layout.

---

## Dimensions

**Total Container:** 1200×1200px
**Tile Grid:** 1200×1200px

**Why 1200px?**
- Fits modern displays (1920×1080+)
- Square aspect (treemap algorithm optimal)
- Predictable tile calculations
