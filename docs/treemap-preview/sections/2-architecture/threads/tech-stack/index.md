# Thread: Tech Stack

Technology choices and dependencies.

---

## Task: [Tech Stack](./tasks/01-tech-stack.md)

**Core:**
- React 19 + TypeScript
- Next.js 15 (static export mode)
- Tailwind CSS v4

**Libraries:**
- d3-hierarchy (treemap layout)
- lucide-react (icons)

**Dev Tools:**
- ESLint + Prettier
- TypeScript strict mode

---

## Tech Rationale

**React 19:** Latest features, Hooks, functional components  
**Next.js 15:** Static export for simple deployment  
**Tailwind v4:** Utility-first CSS, OKLCH color space  
**d3-hierarchy:** Industry-standard treemap algorithm  
**lucide-react:** Modern icon library, tree-shakeable
